# SK1Agent

This is a C# console application using the Microsoft Semantic Kernel framework.

## Getting Started

- Build: `dotnet build SK1Agent/SK1Agent.csproj`
- Run: `dotnet run --project SK1Agent/SK1Agent.csproj`

## Features
- Integrates Microsoft Semantic Kernel for AI agent capabilities.

## Requirements
- .NET 8.0 or later

## Project Structure
- `SK1Agent/` - Main C# project directory
